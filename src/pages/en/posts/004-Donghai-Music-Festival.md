## Good Reads and Learning

- **[CSS Masking](https://web.dev/css-masking/)**: An introduction to the experimental `mask-image` property, including SVG masking and gradient effects. Also check out this [classic article](https://www.zhangxinxu.com/wordpress/2017/11/css-css3-mask-masks/) by Zhang Xinxu.

- **[AVIF: A New Image Format](https://jakearchibald.com/2020/avif-has-landed/)**: Derived from AV1 video, AVIF offers incredible compression. A photo that is 74k as a JPEG and 43k as a WebP is only 18k as an AVIF.

- **[Web APIs for Dual-Screen and Foldable Devices](https://blogs.windows.com/msedgedev/2020/09/14/introducing-dual-screen-foldable-web-apis/)**: Microsoft Edge team introduces two experimental Chromium features: the `screen-spanning` CSS media query and the `getWindowSegments` JS API.

- **[The Mystery of the "Garbled" Text from Deyang](https://zhuanlan.zhihu.com/p/258567705)**: A fascinating look at how a seemingly garbled official notice was actually just encoded text. A great way to learn about [Chinese character encoding](https://zhuanlan.zhihu.com/p/46216008).

- **[Simulating Object Collisions with Canvas](https://www.kffhi.com/blog/2020/08/22/Simulating-Object-Collisions-with-Canvas/)**: A step-by-step guide to building a particle collision simulation from scratch. Original article [here](https://joshbradley.me/object-collisions-with-canvas/).

- **[Machine Learning Foundations Notes](https://juejin.im/post/6859697422605598734)**: Notes from Professor Hsuan-Tien Lin’s "Machine Learning Foundations" course at Taiwan University. Covers [When](https://juejin.im/post/6859697422605598734), [Why](https://juejin.im/post/6861235413085454349), [How](https://juejin.im/post/6874914360964317198), and [Better Learning](https://juejin.im/post/6874908302514323469).

- **[China’s Big Cycle and Its Currency](https://finance.sina.com.cn/china/gncj/2020-09-21/doc-iivhvpwy8038347.shtml)**: Ray Dalio, founder of Bridgewater Associates, analyzes China's history, mindset, and rise since 1949.

## Open Source and Tools

- **[GitHub @vue/lit](https://github.com/yyx990803/vue-lit)**: A tool for writing Web Components using Vue syntax, built with `lit-html` and `@vue/reactivity`. Explained [here](https://juejin.im/post/6874383765070675976).

- **[GitHub Brahmos](https://github.com/brahmosjs/brahmos)**: A UI library that uses React APIs with native templates—JSX without the VDOM. Compare with [Superfine](https://github.com/jorgebucaran/superfine) and [Hyperapp](https://github.com/jorgebucaran/hyperapp).

- **[instant.page: Just-in-Time Preloading](https://instant.page/)**: A single line of code can speed up your site by preloading links when a user hovers over them.

- **[ARWES: Sci-Fi Cyberpunk UI Framework](https://arwes.dev/)**: A futuristic UI framework for web apps. Even the documentation uses ARWES, and it features great sound effects and animations.

- **[GitHub GrapesJS](https://github.com/artf/grapesjs)**: An open-source, no-code framework for building page builders. Check out the [Web Page Demo](https://grapesjs.com/demo.html) and [Newsletter Demo](https://grapesjs.com/demo-newsletter-editor.html).

- **[GitHub Repluggable](https://github.com/wix/repluggable)**: A library for implementing Inversion of Control in frontend apps, making complex project development easier.

- **[filters.css: CSS-Only Filters Library](https://bansal.io/filters-css#blur)**: Apply a wide variety of color filters using pure CSS. Great for image processing workflows.

- **[Vime Player](https://vimejs.com/)**: A customizable, extensible, and framework-agnostic media player.

- **[GitHub Default Branch: Moving from 'master'](https://www.jiqizhixin.com/articles/2020-09-21-10)**: Following the "Black Lives Matter" protests, GitHub is changing the default branch name to be more inclusive.

## iOS and Multi-Platform

- **[An iOS Developer’s Advanced Journey](https://mp.weixin.qq.com/s/cKuVFRPUnnScLM3y26mgIQ)**: Based on a popular V2EX thread, this is a roadmap for any frontend dev looking to master iOS development.

- **[Understanding Podfile Parsing Logic](https://mp.weixin.qq.com/s/f9YcS9eYS_RShLsTXS-SvA)**: Think of `Podfile` as the `package.json` for iOS. This article covers everything from Xcode project structure to fine-grained configuration.

- **[Governing Flutter Binary Sizes](https://mp.weixin.qq.com/s/adC-YUWd-xuUlzeAPHzJoQ)**: Meituan shares their practices and a dynamic delivery solution for keeping Flutter app sizes under control.
