## Technical Insights

**The Evolution of the Web Since 2008**  
<https://mp.weixin.qq.com/s/p4jM_dqQ09gXNp7cTRTvYw>  
A look back at the interesting milestones and shifts in web technology since the birth of Google Chrome.

**Building Smooth and Natural Flutter Pages**  
<https://mp.weixin.qq.com/s/6FLMcE67BCbXvnbFSRgHFg>  
A technical case study from the Xianyu team on how they optimize performance and user experience in their Flutter-based app.

**Building a Programming Language with JavaScript - AST**  
<https://mp.weixin.qq.com/s/JMl8LOsNYlttxO-KyVK-Vw>  
Written by the author of UglifyJS, this series is a masterclass in the concepts behind compilers and syntax trees.

**How We Lost 54k GitHub Stars**  
<https://httpie.io/blog/stardust>  
The HTTPie team shares a heartbreaking (and instructive) story of how a single mistake led to the loss of their primary 54K-star repository.

**Review ES6 to ES12 Features in One Hour**  
<https://juejin.cn/post/7068935394191998990>  
A rapid-fire refresher on all the major JavaScript features theater added over the last few years.

## Trending Open Source

**HowToLiveLonger: A Programmer’s Longevity Guide**  
<https://github.com/geekan/HowToLiveLonger>  
A science-backed guide for developers on how to maintain health and extend lifespan despite a sedentary career.

**My Custom Alfred Theme: "Cosy Dark"**  
<https://gw.alipayobjects.com/os/k/1/Cosy_Dark.alfredappearance>  
I built this last week; you can download and install it directly to give your Alfred a clean, minimal look.

**Cross-Language C++ Calls on Linux**  
<https://tech.meituan.com/2022/04/21/cross-language-call.html>  
Meituan’s approach to solving the multi-language interop problem by wrapping C++ shared libraries for use across different services.

**The Rust Programming Language (Chinese Translation)**  
<https://rustwiki.org/zh-CN/book/>  
A fantastic resource for getting started with Rust. The documentation is incredibly well-structured for rapid learning.

**DOTOWN: Open-Source Pixel Art from Nintendo**  
<https://dotown.maeda-design-room.net/>  
A charming collection of low-bit pixel art assets available for public use.

**Kap: Open-Source Screen Recording for Mac**
