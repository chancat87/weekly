## Front-End Technology

**The Complete Guide to React Re-renders**  
<https://mp.weixin.qq.com/s/SH7N2f5ZhUhysQ7_G2s9rQ>  
What exactly is a re-render? which ones are necessary and which ones are just wasting cycles? if you’ve ever been confused about React performance, this article has the answers.

**React, I Love You, But You’re Bringing Me Down**  
<https://marmelab.com/blog/2022/09/20/react-i-love-you.html>  
A provocative and interesting critique of the current state of the React ecosystem from a developer’s perspective.

**Why Immer Outperforms Immutable.js (90% of the Time)**  
<https://zhuanlan.zhihu.com/p/564029685>  
A technical deep dive into why `immer` is often the better choice for managing immutable state in modern applications.

**iPhone 14 Physical Screen Sizes**  
<https://useyourloaf.com/blog/iphone-14-screen-sizes/>  
A handy reference for developers on the various screen dimensions and safe areas for the entire iPhone 14 lineup.

## Trending Open Source

**iOS 16 Live Activities & Dynamic Island Demos**  
<https://github.com/1998code/iOS16-Live-Activities>  
Several developers have already built some very clever functional demos using Apple’s new ActivityKit and WidgetKit features.

**Web-Clipper: Save the Web to your Notes**  
<https://github.com/webclipper/web-clipper>  
A powerful extension for clipping web content directly to Notion, Bear, Yuque, and more. it’s a great foundation if you’re looking to build your own information archiving tool.

**Qwik: HTML-First Frontend Framework**  
<https://github.com/BuilderIO/qwik>  
A revolutionary framework that delivers only 1KB of JS to start. its "resumability" concept is a fascinating look at the future of web performance.

**Homepage: A Highly Customizable Dashboard**  
<https://github.com/benphelps/homepage>  
Perfect for a personal start page or a monitoring dashboard for all your self-hosted apps.
