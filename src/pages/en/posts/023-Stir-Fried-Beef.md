**json-schema-to-ts: Infer TS Types from JSON Schema**  
<https://github.com/ThomasAribart/json-schema-to-ts>

```javascript
const dogSchema = {
  type: 'object',
  properties: {
    name: { type: 'string' },
    age: { type: 'integer' },
  },
  required: ['name', 'age'],
};
// Automatically infers the TypeScript type!
```
