## Trending Tools

**Mole 1.8 is here**
<https://github.com/tw93/Mole>
Update logs below, mainly focused on various optimizations and capability enhancements.

1. Shell Completion Support: Supports bash/zsh/fish auto-completion, install by running `mo completion`.
2. Zig & Browser Cleanup: Cleans up Zig build artifacts (.zig-cache, zig-out) and old versions of Chrome/Edge.
3. Smart Vendor Protection: Automatically detects and skips Rails vendor directories during purge to prevent accidental deletion of dependencies.
4. Installation Process Enhancement: `install.sh` improved for robustness, supports specifying version/edge installation and `--update`.
5. Testing & Quality: Added 2000+ lines of tests, covering completion, browser cleanup, purge protection, cache cleanup, etc.
6. Bug Fixes: Fixed pnpm Corepack hanging issue, fixed mo purge fd matching issue, improved macOS compatibility documentation.

<https://github.com/pamburus/hl>
