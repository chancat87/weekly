<https://mp.weixin.qq.com/s/jg-_eCA7TH6MP3siOjtVGg>

## Trending Open Source

**SingleFile: Save a Webpage as a Standalone HTML File**  
<https://chrome.google.com/webstore/detail/singlefile/mpiodijhokgodhhofbcjdecpffjipkle>  
Traditional "Save As" usually creates a messy resource folder. SingleFile solves this by bundling everything—CSS, images, and fonts—into one clean HTML file.  
<img src="https://cdn.fliggy.com/upic/ydfv2y.gif" width="800" />

**Lapce: Lightning-Fast Code Editor Written in Rust**  
<https://github.com/lapce/lapce>  
While it’s still in early development and might not replace your daily driver yet, the performance and UI are very impressive.  
<img src="https://gw.alipayobjects.com/zos/k/di/fB8AWl.jpg" width="800" />

**Interactive Engineering Roadmaps**  
<https://roadmap.sh/roadmaps>  
Whether you want to be a professional frontend developer or excel in DevOps, these visual guides show exactly what you need to learn at every stage.  
<img src="https://gw.alipayobjects.com/zos/k/94/NtKzmt.jpg" width="800" />

**Handle: A Chinese version of Wordle**  
<https://handle.antfu.me/>  
A brilliant localization of the Wordle craze for Chinese characters.  
<img src="https://gw.alipayobjects.com/zos/k/cj/Xnmt5I.jpg" width=300/>

**Docusaurus: The Static Site Generator from Meta**  
<https://github.com/facebook/docusaurus>  
One of the most popular tools for building project documentation. it’s fast, accessible, and has great out-of-the-box SEO.
