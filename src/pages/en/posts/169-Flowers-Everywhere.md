## Trending Tools

**BewlyBewly: A Bilibili Beautification Extension**  
<https://chromewebstore.google.com/detail/bewlybewly/bbbiejemhfihiooipfcjmjmbfdmobobp>  
A very polished Chrome extension that gives Bilibili a clean, modern interface with almost no setup required. it might actually make you visit Bilibili more often!  
<img src="https://gw.alipayobjects.com/zos/k/be/SCR-20240317-rtst.png" width="800" />

**Sampler: A Cool Command-Line Tool**  
<https://sampler.dev/>  
A CLI tool that lets you sample any dynamic process directly from your terminal—like watching DB changes, monitoring MQ messages, or getting notifications when deploy scripts finish. Beautifully designed.  
<video width="800px" preload muted autoplay loop><source src="https://gw.alipayobjects.com/os/k/2j/demo.mp4" type="video/mp4"></video>

**Awesome macOS Command Line**  
<https://git.herrbischoff.com/awesome-macos-command-line/about/>  
A list of shell commands and tools specifically for macOS. it includes many hidden configurations that aren't available in the UI, like excluding apps from dark mode, font smoothing, and window transparency.
