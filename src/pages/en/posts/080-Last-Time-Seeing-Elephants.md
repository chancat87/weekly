## Front-End Technology

**The History and Evolution of React Concurrent Rendering**  
<https://mp.weixin.qq.com/s/01sTK6w4BFUzoRc2NKCs1w>

**The Modern JavaScript Tutorial**

## Trending Open Source

**SketchyBar: A Highly Customizable macOS Status Bar**  
<https://github.com/FelixKratz/SketchyBar>  
An incredibly powerful replacement for the standard macOS menu bar. I usually hide my status bar entirely, but this level of customization is very tempting.  
<img src="https://gw.alipayobjects.com/zos/k/i7/K3pQ1k.jpg" width="800" />

**Master Styles: Enhanced Virtual CSS syntax**  
<https://styles.master.co/>  
Just as I was finally getting used to Tailwind, a new contender appears. these utility-first frameworks are all about whether the syntax "clicks" for your brain.  
<img src="https://gw.alipayobjects.com/zos/k/h0/LW0gGu.jpg" width="800" />

**react-embed: Easily Embed Third-Party Widgets**  
<https://github.com/streamich/react-embed>  
A comprehensive set of components for embedding everything from Figma and Gists to YouTube videos and Google Maps directly into your React app.

**GitHub adds Math Support to Markdown**  
<https://github.blog/2022-05-19-math-support-in-markdown/>  
Finally, native TeX-style math support in GitHub readmes and comments. a huge win for technical documentation.  
<img src="https://gw.alipayobjects.com/zos/k/aa/mUTS8x.jpg" width="800" />
