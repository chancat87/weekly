## Technical Related

**The Definitive Guide to JavaScript (7th Ed) Bilingual Online**  
<https://js.okten.cn/>  
For those starting out in frontend development, reading a systematic book is often better than watching scattered videos. here’s an excellent online, bilingual version of the "Rhino Book."

**Deep Dive into CSS :where() and :is()**  
<https://mp.weixin.qq.com/s/HwJuJCVyBvF35FyReIRLqA>  
Browser support for these is actually better than I thought. I’ll be trying them out in my next project.

**iOS Memory Optimization Tools**  
<https://juejin.cn/post/7190296873373007931>  
A guide to detecting and diagnosing memory issues using Xcode, covering everything from memory composition to tools like leaks, vmmap, and malloc_history.

**What to Expect from Vue in 2023**  
<https://thenewstack.io/vue-2023/>  
Evan You discusses the differences between Vue 3 and Vue 2, specifically the evolution of the Virtual DOM.

## Trending Tools

**Image2icon: Rapid App Icon Export**  
<https://img2icnsapp.com/>  
A fantastic tool for app developers to quickly turn images into icons for Mac, iPhone, Favicons, and more, ensuring they meet all platform size requirements.  
<img src="https://gw.alipayobjects.com/zos/k/xx/m83faO.jpg" width="800" />

**NetNewsWire: A Classic RSS Reader**  
<https://netnewswire.com/>  
I’ve been using NetNewsWire for two years now to avoid algorithm-driven feeds. it’s fast, reliable, and looks great on Mac, iPhone, and iPad.  
<img src="https://gw.alipayobjects.com/zos/k/e8/GwN7nw.png" width="800" />

**YouGlish: Master Foreign Pronunciation**  
<https://youglish.com/>  
An incredible tool that lets you search for any word to find YouTube clips of native speakers pronouncing it. it supports multiple languages and is a great way to learn authentic pronunciation in context.  
<img src="https://gw.alipayobjects.com/zos/k/jb/4EvzPd.jpg" width="800" />

**SingleFile: Save a Webpage into a Single HTML**  
<https://chrome.google.com/webstore/detail/singlefile/mpiodijhokgodhhofbcjdecpffjipkle>  
Saving the web as a "folder" often leads to broken images and styles. SingleFile bundles the entire page into a single HTML file that renders perfectly offline.  
<img src="https://cdn.fliggy.com/upic/evpSas.gif" width="800" />
