## Front-End Technology

**Deep Dive: The React useEvent RFC**  
<https://github.com/ascoders/weekly/blob/master/%E5%89%8D%E6%B2%BF%E6%8A%80%E6%9C%AF/240.%E7%B2%BE%E8%AF%BB%E3%80%8AReact%20useEvent%20RFC%E3%80%8B.md>  
`useEvent` aims to solve a classic React problem: how to keep a stable function reference while still accessing the latest state. this breakdown covers the proposal and its potential impact.

**Multi-Screen Window Placement: First Working Draft**  
<https://mp.weixin.qq.com/s/d1IQyMPU7IReD9RB-sdc8g>  
The underlying technology for our multi-screen customer service workspace has finally reached its first public draft. a big win for complex web apps.

**The Ultimate Guide to Regular Expressions**  
<https://segmentfault.com/a/1190000040988916>  
Regex is a headache to write but a lifesaver when it works. this comprehensive tutorial is a great bookmark for when you need a refresher.

## Trending Open Source

**awesome-mac: The Curated List of Mac Software**  
<https://github.com/jaywcjlove/awesome-mac>  
A massive collection of some of the best macOS apps and utilities. perfect for discovering your next favorite tool.

**reveal.js: The HTML Presentation Framework**  
<https://revealjs.com/#/4>  
One of the most powerful ways to build and deliver professional slide decks entirely within the browser. its animations and visuals are world-class.  
<img src="https://gw.alipayobjects.com/zos/k/oc/9Cgbom.jpg" width="800" />

**fresh: Deno’s Modern Web Framework**  
<https://github.com/denoland/fresh>  
A fresh (pun intended) take on web development from the Deno team. it’s high-performance and introduces some very interesting architectural ideas.

**htmx: Extending HTML with Modern Features**  
<https://github.com/bigskysoftware/htmx>  
A brilliant library that lets you access AJAX, CSS Transitions, and more directly in your HTML using attributes. clean, readable, and powerful.

```html
<div hx-post="/mouse_entered" hx-trigger="mouseenter">[Here Mouse, Mouse!]</div>
```
