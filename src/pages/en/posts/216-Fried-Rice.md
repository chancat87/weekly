## Trending Tools

**JetBrains Maple Mono: A combined CJK and Latin font**  
<https://github.com/SpaceTimee/Fusion-JetBrainsMapleMono>  
An open-source user created a combined CJK and Latin font for my favorite programming font, JetBrains Mono, called "JetBrains Maple Mono." It looks great! I chose `JetBrainsMapleMono-NF-NR-XX`; its Chinese characters are more compact than default ones and look very comfortable.  
<img src="https://cdn.fliggy.com/uPic/SCR-20250323-rqzp21.jpeg" width="800" />

**AstroWind Template**  
<https://github.com/onwidget/astrowind>  
If you need to build a product official site, corporate site, or personal site, try AstroWind. Developed using Astro 5.0 + Tailwind CSS, it has all the basic framework elements—perfect for teams without a designer.  
<img src="https://cdn.fliggy.com/uPic/SCR-20250323-sgki21.jpeg" width="800" />

**x-kit: Tool for scraping and analyzing X (Twitter) user data and tweets**  
<https://github.com/xiaoxiunique/x-kit>  
A tool for scraping and analyzing X user data and tweets. It automatically fetches basic info and tweets for specified users, periodically updates timeline data, and supports local storage and GitHub Actions for automation. You can use it for interesting things like displaying your recent tweets on your blog.  
<img src="https://cdn.fliggy.com/uPic/SCR-20250323-sibg21.jpeg" width="800" />

**ZeoSeven Fonts: Free commercial fonts and WebFonts CDN**  
<https://fonts.zeoseven.com/>  
A passion project providing simple ways to embed fonts into your website via CDN. Includes favorites like LXGW WenKai and JetBrains Maple Mono. If you're looking for beautiful fonts, check here.  
<img src="https://cdn.fliggy.com/uPic/SCR-20250323-rxpi21.jpeg" width="800" />

**Wis: A tool for querying domain information**  
<https://w.is/>  
What a short domain name! A tool for querying domain info—very comprehensive.  
<img src="https://cdn.fliggy.com/uPic/SCR-20250323-rwqh21.jpeg" width="800" />
