## Technical Related

**ChatGPT Prompt Engineering for Developers**  
<https://www.deeplearning.ai/short-courses/chatgpt-prompt-engineering-for-developers/>  
Highly recommend this course from Andrew Ng. it’s exceptionally useful for engineers looking to master the art of prompt engineering.

**WebContainers: Node.js on Safari and iPadOS**  
<https://blog.stackblitz.com/posts/webcontainers-are-now-supported-on-safari/>  
WebContainers now run on Safari, meaning you can technically run Node.js directly on your iPad. A must-read for anyone interested in browser-based dev environments.

**WebAssembly Across Environments**  
<https://mp.weixin.qq.com/s/u7xAvi6r8Zqhvv3uRzl-EA>  
Explore how WASM allows different languages to run across various hosts—from running C++ in the browser to running Rust in Node.js.

## Trending Tools

**Free Shared ChatGPT Accounts**  
<https://chat-shared.zhile.io/shared.html>  
A project sharing hundreds of ChatGPT accounts for those who can't easily access the service. A great example of the open-source spirit, though I still recommend using your own account for full functionality.  
<img src="https://gw.alipayobjects.com/zos/k/g4/bGblDF.jpg" width="800" />

**Taira Komori: Free Commercial Sound Effects**  
<https://taira-komori.jpn.org/freesoundcn.html>  
A library of high-quality, free-to-use sound effects—covering nature, animals, combat, and ambient noise. an essential bookmark for game developers.  
<img src="https://gw.alipayobjects.com/zos/k/8q/C0nf81.jpg" width="800" />

**TTSMaker: Online Text-to-Speech**  
<https://ttsmaker.com/zh-cn>  
A powerful text-to-speech engine that supports multiple languages and styles. it's free for commercial use and perfect for video narration.  
<img src="https://gw.alipayobjects.com/zos/k/yl/A1YD6G.jpg" width="800" />

**V2EX Polish: A Modern Look for V2EX**  
<https://v2ex.com/t/935916?p=2>  
V2EX is still one of the best developer communities in China, but its UI is aging. This plugin refreshes the visual experience and adds some great new interactions.  
<img src="https://gw.alipayobjects.com/zos/k/o2/EsXsk2.jpg" width="800" />

**Openverse: A Creative Commons Search Engine**  
<https://openverse.org/zh-cn>  
Search over 700 million reusable creative assets—images, audio, and video—completely for free. it even provides an API for your own projects.  
<img src="https://gw.alipayobjects.com/zos/k/qc/YcyqBH.jpg" width="800" />

**EarthCam: Live Webcams Around the Globe**  
<https://earthcam.com/>  
Watch live streams from cities, zoos, and even aquarium tanks around the world. A fascinating way to see what’s happening globally in real-time.  
<img src="https://gw.alipayobjects.com/zos/k/xi/N6tqv1.jpg" width="800" />

**Pake 2.0: Lightweight Desktop Apps with Rust**  
<https://github.com/tw93/Pake/releases/tag/V2.0.0>  
The major "Mom to Grandma" update!

- Memorize window positions and sizes.
- Support for file downloads and better link handling.
- Command-line support for local static files, custom UserAgents, and tray icons.
