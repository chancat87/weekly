## Trending Open Source

**Helix: A Post-Modern Modal Editor**  
<https://github.com/helix-editor/helix>  
A Rust-based, Vim-inspired editor that’s gaining a lot of traction among experienced terminal users. definitely worth a look if you love modal editing.  
<img src="https://gw.alipayobjects.com/zos/k/j1/3hQpSO.jpg" width="800">

**Textual: TUI Framework for Python**  
<https://github.com/Textualize/textual>  
Textual is a powerful library for creating high-end, responsive terminal user interfaces (TUIs) with Python. it brings modern UI logic to the command line.  
<img src="https://gw.alipayobjects.com/zos/k/6r/HxuZHn.jpg" width="800" />

**Rust Language Bible**  
<https://github.com/sunface/rust-course>  
If you’re looking to pick up a second language, Rust is a fantastic choice. this comprehensive guide is one of the best resources for Chinese-speaking developers.  
<img src="https://gw.alipayobjects.com/zos/k/0s/T70eT4.jpg" width="800" />

**Wolfia: Turn your App into a Link**  
<https://www.wolfia.com/>  
A fascinating concept: upload your APK and run it in a remote virtual machine that you can access instantly via a standard web URL.  
<img src="https://gw.alipayobjects.com/zos/k/s7/FbHDI9.png" width="800" />

**Gimli Tailwind: DevTools for Tailwind CSS**  
<https://chrome.google.com/webstore/detail/gimli-tailwind/fojckembkmaoehhmkiomebhkcengcljl>  
A browser extension that makes inspecting and debugging Tailwind classes directly in Chrome DevTools much more intuitive.  
<img src="https://gw.alipayobjects.com/zos/k/0h/vigGks.png" width="800" />
