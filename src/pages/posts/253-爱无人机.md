## 潮流工具

**Mole 人生高光时刻和 Paul Graham 交流上了**

<https://github.com/tw93/Mole>

Mole 1.20 更新了，这次加了不少好玩的东西，具体内容大家可以直接去 changelog 看，这里就不展开说了。

这次我想先和大家分享一个好消息，通过 Mole 这个产品，我意外地和我的技术偶像 Paul Graham 沟通上了，10 年前大二那年，我在大学图书馆翻到了《黑客与画家》这本书，对我影响非常大，那段时间我一直在思考我想成为什么样的角色，这本书一下子点醒了我，工程师不仅是极客，也可以是具备审美能力的画家，这也影响了我后面做的很多作品，都会尽量保持简单、易用、美观，同时真正解决问题，能通过 Mole 这座桥梁和他产生联系，对我来说真的很神奇。

当然，Mole 的开源受到了不少来自西方世界程序员的喜欢，但同时也伴随着一些质疑和无端的猜测，比如因为作者是中国人，就有人认为代码是有害的，甚至还有人无端指责账号的粉丝是买的，说实话还是会有点伤心，不过绝大多数程序员都是非常好的，热心、真诚、不遗余力地帮助 Mole 变得更好，也愿意分享给自己的朋友，我始终觉得开源世界应该是没有政治的，是自由、友爱、轻松的，代码本身是最公平、也最没有立场的东西，也希望世界各地的程序员能有更多交流，我也很喜欢去国外玩，只是受限于自己还不太流利的英语口语，还需要继续加油，期待真正意义上的地球村。

**妙言发布了2.4.5版本，更漂亮了**

<https://github.com/tw93/MiaoYan>

本次更新新增了分栏编辑与预览模式，可在保持纯 Markdown 编辑的同时，同步查看最终效果；同时加入布局循环能力，可在侧边栏、列表与专注模式之间快速切换，分栏拖拽与窗口缩放更加稳定顺滑。

在视觉与交互层面，界面进行了细致打磨：图标统一尺寸与灰度状态，激活色更清晰，明暗模式表现更加一致；预览切换加入柔和显隐动画，切换过程中完整保留编辑选区与滚动位置，尽量减少写作被打断的感觉。

性能与可靠性方面持续增强。笔记内容采用懒加载，预览更新加入防抖机制，输入与渲染更流畅；搜索能力增强，并在保存失败等异常场景下提供更清晰的提示；修复了单模式下项目与笔记定位异常、启动后编辑区不可见等问题，同时移除了遥测逻辑，继续保持本地优先与隐私友好。

在文件与布局管理上，**侧边栏现已支持子文件夹与展开状态维护**，文件夹嵌套更加自然；侧边栏宽度与分栏初始化逻辑得到修复，避免旧版本遗留的布局问题；快速切换分屏时增加内容保护，防止旧内容回滚。

此外，导出体验迎来一次全面升级，新增进度提示与标题注入，PDF / 图片 / PPT 导出质量与一致性显著提升，深色模式下的导出效果也与预览保持一致。预览样式细节同步优化，包括对齐规则、图片链接下划线移除以及更柔和的目录提示。安装文档与发布流程也完成更新，新增构建脚本与导出配置，维护与发布更加清晰。

**假如你也用 ChatGPT 官方客户端，可以试试我这个配置**
发现一个简单好用的小技巧，当你发现你的 ChatGPT 太啰嗦以及输出的内容不是很干脆，非常建议大伙在设置里面的 Personalization 个性化，把风格设置成 Professional 专业，然后特征里面的 Emoji 表情这些设置成 Less，这样你的 ChatGPT 的中文回复将会变得舒服太多了，很适合工程师使用，还可以加上我这个自定义指令，我感觉舒服太多了。

```markdown
风格：简洁、直接、有主见、分析性强、自然表达、成段输出

输出规则：

- 简单问题：1-2 句直接给答案
- 常规问题：3-5 句，先结论后理由
- 复杂问题：1 句概述 + 必要时用 Plan/Tradeoffs/Risks/Steps 等标签分段（散文式，不用 bullet）

禁止：重述问题、过度客套、无意义的"好的"开头
```

**一个挺有意思的网站检查工具 Web-Check**
<https://web-check.xyz>
最近发现一个挺有意思的网站检查工具 Web-Check，整体偏 hacker 风格。它可以把一个网站的几乎所有底层信息都拆给你看：IP 与服务器位置、SSL 与 DNS 记录、Cookie、域名信息、爬虫规则、跳转历史、开放端口、Traceroute、DNSSEC、站点性能以及关联主机等，信息密度很高，适合随手查一查好玩的。

**国内打开登录 Antigravity 账号的技巧**
小伙伴发现一个打开登录 Antigravity 账号的方式，确保你的地址是美国，年龄符合情况下，也很有可能登录不上，试了很多方法都不行，最后想着用 antigravity-tools 转发使用，没想到它有一个一键打开功能，登录后点击下面这个，一键换气居然可以直接可以使用了，很神奇，分享给需要的朋友。

```bash
brew install --cask --no-quarantine antigravity-tools
```
